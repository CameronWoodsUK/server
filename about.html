About my site

This is a website just for fun that I've made in my free time.
It has a couple of silly little features you can feel free to play around with

The cute animals page

The random animals are found using some tools I found online that return random photos of the animals in question.
The animal of the day is found by taking today's top trending photo from their respective subreddits.

How things work

This website's server runs on code I wrote in Rust more or less from scratch, so if something doesn't work properly that's probably why.
Every request your browser sends is parsed and dealt with using code I wrote myself that's running either on my laptop or a raspberry pi, so some things may take a minute to load, please bear with.
If you're interested you can see the source code here!
