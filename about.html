About my site

This is a website I've made in my free time.
It has a couple of silly little features you play around with.

The cute animals page

The random animals are chosen using some tools I found online that return random photos of the animals in question.
The top animal is found by taking today's top trending photo from their respective subreddits.

The calculator

All the sums you put into the calculator are sent to my assisstant who works them out by hand so please be patient if he takes a minute
He says some weird things sometimes though, so just ignore his ramblings and definitely don't try to send him any messages via the calculator input

How things work

The server hosting this site and all the features and decoration of the site were written entirely by me more or less from scratch, so if something doesn't work properly that's probably why.
Every request your browser sends is dealt with using code I wrote myself that's running on an old raspberry pi, so some things may take a minute to load, please bear with.
If you're interested you can see the source code here!

Proxy server

The code viewer thing was such a massive pain to get to get working and I ended up having to make my own proxy server
If you want to try it out for whatever reason then just append a full link including the http(s) bit after "cameronwoods.lol/proxy/[YOUR LINK HERE]"
